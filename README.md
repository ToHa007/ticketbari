# 🎫 TicketBari – Full Stack Event Ticket Booking Platform

TicketBari is a complete full-stack ticket booking web application built with modern technologies.  
It includes secure authentication, real payment integration, role-based dashboards (User, Vendor, Admin), and protected routes.

## 📦 Installation & Setup Guide

### 1️⃣ Clone Server

git clone https://github.com/yourusername/ticketbari-server.git  
cd ticketbari-server  
npm install  

Create a `.env` file and add required environment variables.

Run Server:

npm start  

---

### 2️⃣ Clone Client

git clone https://github.com/yourusername/ticketbari-client.git  
cd ticketbari-client  
npm install  

Run Development Server:

npm run dev  

Build for Production:

npm run build
